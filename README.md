# geophysics_gravity_test
一维重力异常   
参数设定  
异常深度：200(SI)  
异常半径：20(SI)  
剩余密度：100(SI)   
计算与绘图工具  
Python   
方法公式描述  
重力异常g：重力位对z偏导  
V_zz:重力位对z二次偏导  
V_xz:重力位先后对x、z进行偏导  
V_xy:重力位先后对y、x进行偏导   

数据范围  
y与z均为0  
x范围为300到299（包括300和299），步长为1。   

代码、数据及图像见本文件夹
